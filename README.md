# 物流工程计算工具箱 Logistics Engineering Calculator

[![GitHub Pages](https://img.shields.io/badge/Live-Online-brightgreen)](https://wyzlyno1.github.io/logistics-calculator)

面向物流工程专业的综合性 Web 计算平台，覆盖库存管理、物流规划、需求预测、供应链模拟等核心算法。纯前端实现，无需安装，打开即用。

## 功能模块 Modules

### 库存管理 Inventory Management
| 模块 | 说明 | 核心公式/方法 |
|------|------|-------------|
| 经济订货批量 (EOQ) | 最优订货量、年总成本、再订货点 | Q* = √(2DS/H)，含数量折扣与缺货模型 |
| 经济生产批量 (EPQ) | 自制件最优生产批量 | Q* = √(2DS/[H(1-d/p)]) |
| 安全库存计算 | 三种模型：需求/提前期/综合不确定 | SS = Z·σ·√L |
| ABC 分类分析 | 库存分类 + 帕累托图 | 按年消耗金额排序分类 |

### 物流规划 Logistics Planning
| 模块 | 说明 | 核心方法 |
|------|------|---------|
| 重心法选址 | 单设施最优选址 | 基本重心法 + 迭代优化 |
| 选址案例分析 | 多方法选址决策对比 | 因素评分法、盈亏平衡法 |
| 运输问题求解 | 产销平衡运输问题 | 最小元素法、伏格尔法、位势法、闭回路法 |
| 物流网络优化 | 图论算法可视化 | Dijkstra、最小生成树、最大流 |

### 需求管理 Demand Management
| 模块 | 说明 | 核心方法 |
|------|------|---------|
| 需求预测 | 时间序列预测 + 误差分析 | 移动平均、加权移动平均、指数平滑 |
| 牛鞭效应演示 | 多级供应链需求放大模拟 | 四级供应链动态仿真 |

### 工具 Utilities
| 模块 | 说明 |
|------|------|
| 报告生成 | 一键导出计算结果为可打印报告 |

## 技术栈 Tech Stack

- **前端**: HTML5 + CSS3 + 原生 JavaScript (ES6+)
- **图表**: Canvas API（帕累托图、网络图、仿真曲线）
- **部署**: GitHub Pages（静态托管，免费）
- **依赖**: 零外部依赖，无框架，无后端
