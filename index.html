关于工具

本工具为物流工程课程学习辅助工具，覆盖库存管理、物流规划、需求预测等核心计算模块。

所有计算均基于物流工程教材中的标准公式，提供逐步求解过程，方便学习与作业参考。

纯前端实现，无需网络，打开即用。

技术说明

HTML + CSS + 原生 JavaScript，无框架、无后端、无数据收集。

所有计算在浏览器本地完成，输入数据不会被上传或存储。

自由使用、分享、修改。用于课程作业时请注明工具来源。
